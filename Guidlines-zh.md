# 微软 REST API 指南

## 1. 简介
作为设计原则，微软 REST API 指南鼓励应用程序开发人员通过 RESTful HTTP 接口访问资源。
为了按照 微软 REST API 指南为平台上的开发人员提供尽可能流畅的体验，REST API 应遵循一致的设计指南，以便轻松、直观地使用它们。

本文档确立了 微软 REST API 应遵循的准则，以便一致地开发 RESTful 接口。
